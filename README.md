# 宏志助航 自动学习助手 (真该死啊)

这是一个基于 Playwright 与 Python 开发的高级自动化脚本，专门用于“宏志助航”学习平台的视频自动播放、防沉迷弹窗自动点击以及**多课程并发监控**。

##  核心功能

*   **多标签页并行监控**: 支持同时打开多个课程视频（Tab），脚本会自动轮询并维护所有页面的播放状态。
*   **原生浏览器调用**: 优先调用系统已安装的 **Google Chrome** 或 **Microsoft Edge**，完美解决视频播放器加载不出（解码器缺失）的问题。
*   **高级抗检测 (Stealth)**: 集成 `playwright-stealth` 指纹伪装，绕过自动化检测。
*   **面向对象架构 (OOP)**: 代码模块化，稳定性极高，支持自适应频率心跳监控。
*   **持久化登录**: 自动保存登录状态到 `user_data`，首次登录后后续可免扫码。

##  环境准备

确保你电脑已安装 Python 3.8 或更高版本。

### 1. 安装依赖
打开终端（CMD 或 PowerShell），进入脚本目录并运行：
```powershell
pip install playwright playwright-stealth
playwright install chromium
```

### 2. 浏览器建议
脚本会自动寻找系统中的 **Google Chrome**（推荐）或 **Microsoft Edge**。如果两者都没有，请运行 `playwright install chromium`。

##  如何运行

在终端中切换到项目目录，输入以下命令启动：
```powershell
python -u main.py
```

##  使用技巧

1.  **首选登录**: 脚本启动后会弹窗，请先手动完成扫码登录。
2.  **多开课程**:
    *   在课程列表页，**右键点击课程 -> “在新标签页中打开”**。
    *   你可以同时打开 3-5 个不同课程的视频页。
3.  **全自动接管**: 脚本会识别到 `Tab-0`, `Tab-1` 等，并自动轮流点击视频中心的播放按钮和出现的“继续学习”弹窗。

##  注意事项与风险 (重要)

> [!TIP]
> **终端乱码修复**: 脚本已内置 UTF-8 修复逻辑。如果你的终端依然显示乱码，请在启动脚本前先运行命令 `chcp 65001`。
